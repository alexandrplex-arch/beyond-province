Правила BeyMarket

Основные правила и политика платформы

Основные правила

  • Запрещена продажа запрещенных законом товаров и услуг
  • Не допускается мошенничество и обман пользователей
  • Запрещено размещение контента сексуального или порнографического характера
  • Нельзя публиковать личную информацию других пользователей без их согласия
  • Запрещены оскорбления, угрозы и дискриминационные высказывания
  • Не допускается спам и навязчивая реклама

Правила торговли

  • Продавец обязан предоставить полную и достоверную информацию о товаре
  • Цена должна быть указана корректно и не может быть изменена после подтверждения сделки
  • Фотографии товара должны соответствовать действительности
  • Запрещена продажа одного товара нескольким покупателям одновременно
  • При отмене сделки по инициативе продавца, покупателю должны быть возвращены все средства
  • Все споры решаются через службу поддержки платформы

Правила общения

  • Общайтесь уважительно с другими пользователями
  • Не переходите на личности в ходе обсуждения товаров
  • Запрещены попытки обойти систему безопасности платформы
  • Сообщения должны быть по теме объявления
  • Не злоупотребляйте функцией жалоб на других пользователей
  • При нарушении правил чата, пользователь может быть заблокирован

Безопасность

  • Никогда не передавайте свои учетные данные третьим лицам
  • Используйте сложные пароли для защиты аккаунта
  • Внимательно проверяйте информацию о продавце перед сделкой
  • Сообщайте администрации о подозрительных действиях
  • Не переходите по подозрительным ссылкам от других пользователей
  • Храните скриншоты важных переговоров и соглашений

Санкции за нарушения

  • Первое нарушение - предупреждение
  • Повторное нарушение - временная блокировка аккаунта (3 дня)
  • Систематические нарушения - постоянная блокировка аккаунта
  • Мошенничество - немедленная блокировка без возможности восстановления
  • Нарушение законодательства - блокировка и передача данных правоохранительным органам
